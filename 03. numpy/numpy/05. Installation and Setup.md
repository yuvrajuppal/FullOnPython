

If you are using a standard Python environment, you will need to install NumPy before you can use it.

**Installation** Open your terminal or command prompt and run one of the following commands:

- **Using pip** (the standard Python package manager):
    
    Bash
    
    ```
    pip install numpy
    ```
    
- **Using conda** (if you are using the Anaconda distribution):
    
    Bash
    
    ```
    conda install numpy
    ```
    

**Setup (Importing)** Once installed, you need to bring it into your Python script or Jupyter Notebook. The universal standard is to import NumPy under the alias `np`.

Python

```
import numpy as np

# Verify the installation by printing the version
print(np.__version__) 
```
