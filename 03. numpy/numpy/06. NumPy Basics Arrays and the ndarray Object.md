

The core of NumPy is the **`ndarray`** (N-dimensional array) object. Think of it as a grid of values, where all values must be of the exact same data type (e.g., all integers, or all floats).

#### **A. Creating Arrays from Python Lists**

You can create a NumPy array by passing a standard Python list into the `np.array()` function.



```Python
import numpy as np

# Creating a 1-Dimensional array (like a standard list)
list_1d = [1, 2, 3, 4, 5]
array_1d = np.array(list_1d)
print("1D Array:\n", array_1d)

# Creating a 2-Dimensional array (like a matrix or table)
list_2d = [[1, 2, 3], [4, 5, 6]]
array_2d = np.array(list_2d)
print("\n2D Array:\n", array_2d)
```

#### **B. Creating Arrays with Built-in NumPy Functions**

Often, you need to create arrays of a certain size from scratch without typing out lists. NumPy has several built-in methods for this:



```Python
# Create an array of 5 zeros
zeros_array = np.zeros(5)
print("Zeros:\n", zeros_array)

# Create a 2x3 matrix filled with ones
ones_matrix = np.ones((2, 3))
print("\nOnes:\n", ones_matrix)

# Create an array of numbers from 0 to 9 (similar to Python's range)
range_array = np.arange(10)
print("\nArange:\n", range_array)

# Create an array of 5 evenly spaced numbers between 0 and 1
linspace_array = np.linspace(0, 1, 5)
print("\nLinspace:\n", linspace_array)

# Create a 3x3 matrix filled with random numbers between 0 and 1
random_matrix = np.random.rand(3, 3)
print("\nRandom:\n", random_matrix)
```

#### **C. Inspecting the `ndarray` (Key Attributes)**

Once you have an array, you often need to check its size, shape, or data type. The `ndarray` object comes with several built-in attributes to inspect this metadata.


```Python
# Let's use our 2D array from earlier
my_array = np.array([[10, 20, 30], [40, 50, 60]])

# 1. .ndim: The number of dimensions (axes)
print("Dimensions:", my_array.ndim)  # Output: 2

# 2. .shape: The size of the array in each dimension (rows, columns)
print("Shape:", my_array.shape)      # Output: (2, 3)

# 3. .size: The total number of elements in the array
print("Total size:", my_array.size)  # Output: 6

# 4. .dtype: The data type of the elements inside the array
print("Data type:", my_array.dtype)  # Output: int64 (or int32 depending on your system)
```
