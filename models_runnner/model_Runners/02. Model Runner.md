Running Ollama in Python is incredibly straightforward using their official Python library. Here is the step-by-step guide to getting it up and running.

### Step 1: Install the Ollama App

Before Python can talk to Ollama, you need the core software running on your computer.

1. Go to [ollama.com](https://ollama.com/) and download the app for your operating system (Windows, macOS, or Linux).
    
2. Install and open the application. It will run in the background.
    
3. Open your computer's terminal or command prompt and pull a model to use. For example, to download Llama 3, run:
    
    Bash
    
    ```
    ollama run llama3
    ```
    ```
    ollama run llama3.2:1b
    ```
    _(Note: This might take a few minutes depending on your internet connection, as the model files are several gigabytes)._
    

### Step 2: Install the Python Library

Once Ollama is installed and you've downloaded a model, open your terminal and install the official Python package:

Bash

```
pip install ollama
```

### Step 3: Write the Python Code

Here is how you can use Python to chat with the model you just downloaded.

#### Basic Chat (Waits for the full response)

This script sends a message to the model and prints the complete answer once it finishes thinking.

```Python
import ollama

# Send a prompt to the model
response = ollama.chat(model='llama3', messages=[
  {
    'role': 'user',
    'content': 'Explain quantum computing in one simple sentence.',
  },
])

# Print the model's reply
print(response['message']['content'])
```

#### Streaming Chat (Prints word-by-word)

If you want a more interactive, ChatGPT-like experience where the text appears as it's being generated, you can use the `stream=True` parameter:

```Python
import ollama

print("Ollama: ", end="", flush=True)

# The stream=True argument makes the model return chunks of text
stream = ollama.chat(
    model='llama3',
    messages=[{'role': 'user', 'content': 'Write a short haiku about coding.'}],
    stream=True,
)

# Print each chunk as it arrives
for chunk in stream:
  print(chunk['message']['content'], end='', flush=True)
  
print() # Add a final newline
```

### Troubleshooting Tips

- **"Connection Refused" Error:** This usually means the Ollama background application isn't running. Make sure you've opened the Ollama app on your computer before running your Python script.
    
- **"Model not found" Error:** You must download the model via the terminal (e.g., `ollama pull mistral` or `ollama pull llama3`) before you can call it in your Python code.
