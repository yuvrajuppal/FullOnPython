### 1. Installation

Just like you run `npm install axios`, you need to install `requests` via Python's package manager, `pip`:

Bash

```
pip install requests
```

### 2. The Basic GET Request

In Axios, you are used to dealing with Promises and `res.data`. Python's `requests` is synchronous by default, meaning the code pauses until the data comes back, so there are no Promises or `.then()` blocks to worry about.

**Your JS Brain (Axios):**



```JavaScript
const response = await axios.get('https://api.example.com/users/1');
console.log(response.data);
```

**Your New Python Code (`requests`):**



```Python
import requests

response = requests.get('https://api.example.com/users/1')
data = response.json() # This converts the JSON response into a Python dictionary

print(data)
```

### 3. Making a POST Request (Sending Data)

Axios automatically handles stringifying JSON when you pass an object. Python's `requests` does the exact same thing if you use the `json=` parameter.

**Your JS Brain (Axios):**



```JavaScript
const response = await axios.post('https://api.example.com/users', {
  name: 'John Doe',
  role: 'Admin'
});
```

**Your New Python Code (`requests`):**



```Python
import requests

payload = {
    "name": "John Doe",
    "role": "Admin"
}

# Use the 'json' parameter, and it automatically sets the correct headers!
response = requests.post('https://api.example.com/users', json=payload)

print(response.status_code) # e.g., 201
```

### 4. What if you _need_ Async? (Meet `httpx`)

The one major difference is that Axios is entirely asynchronous, while `requests` is synchronous (blocking). If you are building an async Python app (using `async def` and `await`) and want an exact Axios equivalent, you should look into a library called **`httpx`**.

It works almost identically to `requests` but supports modern async/await syntax:

Bash

```
pip install httpx
```



```Python
import httpx
import asyncio

async def fetch_data():
    async with httpx.AsyncClient() as client:
        response = await client.get('https://api.example.com/users/1')
        print(response.json())

asyncio.run(fetch_data())
```

### Summary for your JS-to-Python transition:

- `axios.get()` ➔ `requests.get()`
    
- `res.data` ➔ `response.json()`
    
- `res.status` ➔ `response.status_code`
    
- `axios.post(url, data)` ➔ `requests.post(url, json=data)`
