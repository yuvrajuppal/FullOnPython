In Python, “condition operators” usually mean comparison operators (also called relational operators) that compare two values and return `True` or `False`.[](https://www.w3schools.com/python/gloss_python_comparison_operators.asp)

These are used in `if`, `elif`, `while`, and other conditions.[](https://www.tutorialspoint.com/python/python_comparison_operators.htm)

---

## 6 main comparison operators

| Operator | Meaning                  | Example (`a=5`, `b=3`) |
| -------- | ------------------------ | ---------------------- |
| `==`     | Equal to                 | `a == b` → `False`     |
| `!=`     | Not equal to             | `a != b` → `True`      |
| `>`      | Greater than             | `a > b` → `True`       |
| `>=`     | Greater than or equal to | `a >= 5` → `True`      |
| `<`      | Less than                | `a < b` → `False`      |
| `<=`     | Less than or equal to    | `a <= 5` → `True`      |
example 1
```python
a = 10
b = 5

print(a == b)    # False (10 is not equal to 5)
print(a != b)    # True  (10 is not equal to 5)
print(a > b)     # True
print(a >= 10)   # True
print(a < b)     # False
print(a <= 15)   # True
```

example 2
```python
name1 = "Goose"
name2 = "goose"
name3 = "Apple"

print(name1 == name2)      # False (capital G vs lowercase g)
print(name1 == "Goose")    # True
print(name1 != name3)      # True
print(name1 < name3)       # False (G comes after A)
print(name1 > "Apple")     # True
print(name1 <= "Goose")    # True

```
