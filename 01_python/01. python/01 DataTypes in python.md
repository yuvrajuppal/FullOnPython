Here are the four basic data types in Python: `int`, `float`, `str`, and `bool`.

## 1. int (Integer)
```python
age = 25                # int
count = -10             # negative int
big_number = 1234567890 # big int
print(type(age))        # <class 'int'>
```


## 2. float (Floating-point number)
```python
price = 99.99
temperature = -3.5
pi = 3.14159
print(type(pi))         # <class 'float'>
```


## 3. str (String / Text)
```python
name = "Goose"
message = "Hello World"
multiline = """
This is a
multiline
string.
"""
print(type(name))       # <class 'str'>
```

## 4. bool (Boolean)
```python
is_student = True
is_logged_in = False
is_even = (10 % 2 == 0)  # True
print(type(is_student))  # <class 'bool'>
```
