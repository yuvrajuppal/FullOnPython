Pythonâ€™s conditional expression (also called the ternary operator) lets you write a simple `if-else` in a single line, in the form `value_if_true if condition else value_if_false`.

## Basic syntax

The ternary operator has this structure:
```
value_if_true if condition else value_if_false
```

- `condition` is a boolean expression (like `x > 5`, `name == "admin"`, etc.).
- If `condition` is `True`, the whole expression returns `value_if_true`.
- If `condition` is `False`, it returns `value_if_false`.

## Simple example

```python
age = 20
status = "adult" if age >= 18 else "minor"
print(status)  # Output: adult
```

This is equivalent to the longer `if-else`:
```python
age = 20
if age >= 18:
    status = "adult"
else:
    status = "minor"
```

## Practical use cases

##  Assigning a value conditionally
```python
light = "green"
action = "move" if light == "green" else "stop"
print(action)  # Output: move
```
