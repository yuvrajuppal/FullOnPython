Python has three logical (Boolean) operators: `and`, `or`, and `not`, which are used to combine or invert conditions and return `True` or `False`.
## `and` operator

The `and` operator returns `True` only if **both** conditions are `True`; otherwise, it returns `False`.

Truth table for `and`:
- `True and True` → `True`
- `True and False` → `False`
- `False and True` → `False`
- `False and False` → `False`
```python
a = 5
b = 10
print(a > 3 and b > 8)   # True (both conditions are True)
print(a > 7 and b > 8)   # False (first condition is False)
```

`and` is commonly used in `if` statements:
```python
age = 20
marks = 85
if age >= 18 and marks >= 80:
    print("Eligible for scholarship")
else:
    print("Not eligible")
```

## `or` operator

The `or` operator returns `True` if **at least one** condition is `True`; it returns `False` only if both are `False`.
Truth table for `or`:
- `True or True` → `True`
- `True or False` → `True`
- `False or True` → `True`
- `False or False` → `False`
```python
marks = 75
attendance = 60
if marks >= 80 or attendance >= 75:
    print("Passed")
else:
    print("Failed")
```



## `not` operator

The `not` operator reverses the truth value of a condition: `not True` becomes `False`, and `not False` becomes `True`.
Truth table for `not`:
- `not True` → `False`
- `not False` → `True`​
**Example:**

```python
is_adult = False

print(not is_adult)  # True (because is_adult is False)
```

```python
name = ""
if not name:
    print("Name is empty or not provided")
```
