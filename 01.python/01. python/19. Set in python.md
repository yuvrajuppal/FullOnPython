`Python sets are unordered collections of unique, immutable elements, ideal for membership testing and removing duplicates.

## Set Characteristics

Sets store unique itemsâ€”no duplicates allowed. They're mutable (you can add/remove elements) but unindexed and unordered, so you can't access elements by position. Elements must be hashable (like strings, numbers, tuples) but not lists or dictionaries.

## Creating Sets

Create sets using curly braces `{}` or the `set()` constructor.

```python
# Basic set
fruits = {"apple", "banana", "apple"}  # Duplicates auto-removed
print(fruits)  # {'apple', 'banana'}

# Empty set (use set(), not {})
empty = set()

# From list/iterable
numbers = set([1, 2, 3, 2])  # {1, 2, 3}
```


## Common Set Methods

## Add/Remove Elements

**add(x)**: Adds one element.
```python
fruits = {"apple"}
fruits.add("banana")
print(fruits)  # {'apple', 'banana'}
```

**remove(x)**: Removes element (raises error if missing).
```python
fruits.remove("apple")
print(fruits)  # {'banana'}
```
**discard(x)**: Removes without error.
**pop()**: Removes/returns random element.
**clear()**: Empties the set.

**`len()`**
```python
fruits = {"apple", "banana", "apple", "cherry"}
```
