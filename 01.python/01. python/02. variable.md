In Python, a variable is like a named storage box where you can store a value (like a number, text, or `True`/`False`) so that you can use it later in your code.



```python
name = "Goose"
age = 20
price = 99.99
is_active = True
print(name)        # Output: Goose
print(age + 10)    # Output: 30
print(is_active)   # Output: True
```

## Rules for naming variables in Python
Python has strict rules about what variable names are allowed:
1. **Must start with a letter or underscore**
    - OK: `age`, `_name`, `myVar`
    - Not OK: `2name`, `123count` (canâ€™t start with a digit).[](https://www.w3schools.com/python/python_variables_names.asp)

2. **Can only contain letters, numbers, and underscore**    
    - OK: `user_name`, `score1`, `total_price`
    - Not OK: `user-name`, `my var`, `price$` (spaces and special characters like `-`, `$`, `@` are not allowed).
3. **Case-sensitive**
    - `name`, `Name`, and `NAME` are three different variables.
    - So be consistent with uppercase/lowercase.
4. **Cannot use Python keywords (reserved words)**
    - Keywords like `if`, `else`, `for`, `while`, `def`, `class`, `return`, `True`, `False`, `None`, `and`, `or`, `in`, `import` are not allowed as variable names.
    - Example: `if = 10` will give a syntax error.
5. **No spaces in variable names**
    - Names like `my variable` or `first name` are invalid.
    - Use `my_variable` or `first_name` instead.
