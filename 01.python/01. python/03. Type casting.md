Type casting (or type conversion) in Python means changing the data type of a value from one type (like `int`, `float`, `str`, `bool`) to another.


## 1. `int()` — Convert to integer
```python
x = int(5.9)      # 5.9 → 5
y = int("10")     # "10" → 10
z = int(True)     # True → 1
w = int(False)    # False → 0

print(x)  # 5
print(y)  # 10
print(type(x))  # <class 'int'>
```

```python
int("abc")    # ❌ Error
int("3.14")   # ❌ Error (use float() first)
```

## 2. `float()` — Convert to floating point

```python
x = float(5)       # 5 → 5.0
y = float("3.14")  # "3.14" → 3.14
z = float("10")    # "10" → 10.0
w = float(True)    # True → 1.0
v = float(False)   # False → 0.0

print(x)  # 5.0
print(y)  # 3.14
print(type(x))  # <class 'float'>

```

```python
float("hello")  # ❌ Error
```


## 3. `str()` — Convert to string

```python
x = str(42)         # 42 → "42"
y = str(3.14)       # 3.14 → "3.14"
z = str(True)       # True → "True"
w = str(None)       # None → "None"

print(x)  # "42"
print(x + " is the answer")  # "42 is the answer"
print(type(x))  # <class 'str'>
```


```python
level = 15
message = "Current level: " + str(level)  # "Current level: 15"
print(message)
```

## 4. `bool()` — Convert to boolean

```python
b1 = bool(1)        # True
b2 = bool(0)        # False
b3 = bool(3.14)     # True
b4 = bool(0.0)      # False

b5 = bool("hello")  # True
b6 = bool("")       # False
b7 = bool("0")      # True (non‑empty string)

b8 = bool([1, 2])   # True (non‑empty list)
b9 = bool([])       # False (empty list)
b10 = bool(None)    # False

print(b1, b2, b3, b4)  # True False True False
print(b5, b6, b7)      # True False True
print(b8, b9, b10)     # True False False
```
