Aggregate functions (also known as reduction functions) take an array of numbers and "squash" them into a single summary value (or a smaller set of values).

In NumPy, these functions are extremely fast because they are implemented in C.

---

## 1. Common Aggregate Functions

Here are the most frequently used functions:

|Function|Description|
|---|---|
|**`np.sum()`**|Adds all elements|
|**`np.mean()`**|Calculates the arithmetic average|
|**`np.max()` / `np.min()`**|Finds the largest/smallest value|
|**`np.std()`**|Standard deviation (spread of data)|
|**`np.var()`**|Variance|
|**`np.median()`**|The middle value|
|**`np.argmin()` / `np.argmax()`**|Returns the **index** of the min/max value|


---

## 2. The `axis` Parameter (Crucial!)

When working with 2D arrays (matrices), you often don't want the sum of the _whole_ thing. You might want the sum of each column or each row. This is controlled by the `axis` argument.

- **`axis=0`**: Operations happen **down the columns**.
    
- **`axis=1`**: Operations happen **across the rows**.
    

---

## 3. Code Example

```Python
import numpy as np

# Create a 2x3 matrix
# [[1, 2, 3],
#  [4, 5, 6]]
arr = np.array([[1, 2, 3], [4, 5, 6]])

# 1. Global Aggregation (All elements)
print(np.sum(arr))   # Output: 21
print(np.max(arr))   # Output: 6

# 2. Column-wise Aggregation (Collapse rows)
# [1+4, 2+5, 3+6]
print(np.mean(arr, axis=0)) 
# Output: [2.5, 3.5, 4.5]

# 3. Row-wise Aggregation (Collapse columns)
# [1+2+3, 4+5+6]
print(np.sum(arr, axis=1))  
# Output: [6, 15]
```

---

## 4. Handling Missing Data (NaNs)

Standard aggregate functions will return `NaN` (Not a Number) if your array contains even one missing value. NumPy provides "NaN-safe" versions that ignore missing values:

```Python
data = np.array([1, 2, np.nan, 4])

print(np.mean(data))      # Output: nan (fails)
print(np.nanmean(data))   # Output: 2.33 (ignores the nan)
```

**Common NaN-safe versions:** `np.nansum()`, `np.nanmin()`, `np.nanmax()`, `np.nanstd()`.
