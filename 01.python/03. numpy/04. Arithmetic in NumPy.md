Arithmetic in NumPy is where the library truly shines. Unlike Python lists, where `+` joins (concatenates) two lists together, NumPy performs **element-wise** operations. This means the operation is applied to every individual number in the array simultaneously.

---

## 1. Basic Element-wise Operations

If two arrays have the same shape, you can add, subtract, multiply, and divide them directly.


```Python
import numpy as np

a = np.array([10, 20, 30, 40])
b = np.array([1, 2, 3, 4])

# Addition
print(a + b)  # [11, 22, 33, 44]

# Subtraction
print(a - b)  # [9, 18, 27, 36]

# Multiplication (Element-wise)
print(a * b)  # [10, 40, 90, 160]

# Division
print(a / b)  # [10.0, 10.0, 10.0, 10.0]

# Exponentiation (Power)
print(b ** 2) # [1, 4, 9, 16]
```

---

## 2. Scalar Operations

You can also perform arithmetic between an array and a single number (a scalar). NumPy "stretches" the scalar so it applies to every element.


```Python
arr = np.array([1, 2, 3])

print(arr + 5)  # [6, 7, 8]
print(arr * 10) # [10, 20, 30]
```

---

## 3. Broadcasting

This is a powerful NumPy concept. If you have two arrays of different shapes, NumPy will try to "broadcast" the smaller one to match the larger one so the math still works.

For example, if you add a 1D array to a 2D matrix:

```Python
matrix = np.array([[1, 2, 3], 
                   [4, 5, 6]])
row = np.array([10, 20, 30])

# The row [10, 20, 30] is added to BOTH rows of the matrix
result = matrix + row
# [[11, 22, 33],
#  [14, 25, 36]]
```

## 4. Universal Functions (ufuncs)

NumPy provides built-in mathematical functions that are optimized for arrays. These are often faster and more readable than standard arithmetic.

|Function|Description|
|---|---|
|`np.add(a, b)`|Same as `a + b`|
|`np.sqrt(a)`|Square root of every element|
|`np.exp(a)`|Exponential (ex)|
|`np.log(a)`|Natural log|
|`np.sin(a)`|Sine of every element|

## 5. Summary Table: List vs. Array

|Operation|Python List `[1, 2] + [3, 4]`|NumPy Array `np.array([1, 2]) + [3, 4]`|
|---|---|---|
|**`+`**|Concatenates: `[1, 2, 3, 4]`|Adds: `[4, 6]`|
|**`*`**|Repeats: `[1, 2, 1, 2]`|Multiplies: `[3, 8]`|
