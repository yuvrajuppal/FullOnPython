## 2. What is NumPy?

Think of NumPy as the "Math Engine" for Python. While standard Python lists are flexible, they are slow for big data. NumPy introduces the **ndarray** (n-dimensional array), which is a block of memory that allows for lightning-fast calculations.

### Why use it?

- **Speed:** It’s written in C, making it significantly faster than standard Python loops.
    
- **Vectorization:** You can perform operations on entire arrays at once without writing `for` loops.
    
- **Broadcasting:** It has "smart" ways of handling operations between arrays of different shapes.
    

---

## 3. Practical Example: The Power of Arrays

In standard Python, adding two lists requires a loop. In NumPy, it’s a simple addition.



```Python
import numpy as np

# 1. Creating an array (1D and 2D)
simple_list = [1, 2, 3, 4, 5]
n_array = np.array(simple_list)

matrix = np.array([[1, 2], [3, 4]])

# 2. Vectorized Math (No loops needed!)
# Let's double every number in the array
doubled = n_array * 2

# 3. Useful Functions
mean_val = np.mean(n_array)   # Average
sum_val = np.sum(n_array)     # Total sum
sq_root = np.sqrt(n_array)    # Square root of each element

print(f"Original: {n_array}")
print(f"Doubled:  {doubled}")
print(f"Average:  {mean_val}")
```

## 4. Key Concepts to Remember

|Feature|Python List|NumPy Array|
|---|---|---|
|**Data Types**|Can mix types (int, string, etc.)|**Must** be the same type (usually float or int)|
|**Performance**|Slower for large datasets|Extremely fast|
|**Functionality**|Basic (append, pop)|Advanced (Linear algebra, Fourier transforms)|
