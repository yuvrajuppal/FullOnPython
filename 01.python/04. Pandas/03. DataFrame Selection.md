This is the perfect next step. If you have a massive DataFrame, you rarely want to look at the whole thing at once. You want to slice it, dice it, and extract exactly what you need.

In Pandas, selecting data essentially boils down to asking your DataFrame questions like, "Can I see just the names?" or "Who scored higher than a 90?"

Let's use our student DataFrame from the previous example to explore the **four main ways** to select data.

### **The Setup**

Here is our starting data so you can follow along:



```Python
import pandas as pd

student_data = {
    "Name": ["Alice", "Bob", "Charlie", "David"],
    "Age": [20, 21, 19, 22],
    "Major": ["Biology", "Math", "Physics", "Art"],
    "Score": [95, 82, 88, 91]
}

df = pd.DataFrame(student_data)
# Let's set the "Name" column to act as our index (row labels) 
# to make row selection easier to understand.
df = df.set_index("Name") 

print(df)
```

**Output:**



```Plaintext
         Age    Major  Score
Name                        
Alice     20  Biology     95
Bob       21     Math     82
Charlie   19  Physics     88
David     22      Art     91
```

---

### **1. Selecting Columns (The Basics)**

If you just want one or more columns, you use square brackets `[]` and the name of the column.

- **Single Column (Returns a Series):**
    
    
    
    ```Python
    print(df["Major"])
    ```
    
- **Multiple Columns (Returns a smaller DataFrame):** _Note the double brackets `[[]]` because you are passing a list of column names!_
    
    
    ```Python
    print(df[["Major", "Score"]])
    ```
    

### **2. Selecting Rows by Label (`loc`)**

The `loc` (location) property is used when you want to find a row based on its **label** (the index name).


```Python
# Get all of Bob's information
print(df.loc["Bob"])
```

**Output:**

Plaintext

```
Age        21
Major    Math
Score      82
Name: Bob, dtype: object
```

### **3. Selecting Rows by Number (`iloc`)**

The `iloc` (integer location) property is used when you want to find a row based on its **numerical position**, starting at 0. You use this when you don't care about the labels and just want, for example, the very first row.


```Python
# Get the 1st row (Alice)
print(df.iloc[0])

# Get the 1st through the 3rd row (Alice, Bob, Charlie)
print(df.iloc[0:3]) 
```

### **4. Conditional Selection (Filtering)**

This is the most powerful and common way to select data. You pass a condition (like greater than, equal to) inside the brackets, and Pandas returns only the rows where that condition is True.


```Python
# Step 1: Write the condition
condition = df["Score"] > 85

# Step 2: Pass it into the DataFrame brackets
top_students = df[condition]

print(top_students)
```

**Output:**



```Python
         Age    Major  Score
Name                        
Alice     20  Biology     95
Charlie   19  Physics     88
David     22      Art     91
```

_(Bob is excluded because his score is 82)._

**Pro-tip:** Most developers write conditional selections in a single, streamlined line of code like this: `df[df["Score"] > 85]`.
