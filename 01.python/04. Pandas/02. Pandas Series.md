### **What is a Pandas Series?**

Think of a Series as a **single column in a spreadsheet** or a souped-up Python list.

Technically, it is a one-dimensional array that can hold any type of data (integers, strings, decimals, etc.). What makes a Series special is that every piece of data is assigned a label, collectively known as the **index**.

A Series has two main components:

- **The Values:** The actual data you are storing.
    
- **The Index:** The labels for that data. If you don't provide an index, Pandas will automatically assign numbers starting from 0 (just like a regular Python list).
    

---

### **Examples in Action**

To use a Series, you first need to import the Pandas library.

#### **1. Creating a Series with an automatic index**

If you just pass a list of items, Pandas numbers them for you.



```Python
import pandas as pd

# Creating a list of data
my_data = ["Apples", "Bananas", "Cherries"]

# Converting it to a Series
my_series = pd.Series(my_data)

print(my_series)
```

**Output:**



```Plaintext
0      Apples
1     Bananas
2    Cherries
dtype: object
```

#### **2. Creating a Series with a custom index**

You can define your own labels, which makes retrieving data much easier. You can do this by passing an `index` list, or by simply converting a Python dictionary into a Series.



```Python
import pandas as pd

# Using a dictionary where keys become the index and values become the data
student_scores = {
    "Alice": 95,
    "Bob": 82,
    "Charlie": 88
}

scores_series = pd.Series(student_scores)

print(scores_series)
```

**Output:**



```Plaintext
Alice      95
Bob        82
Charlie    88
dtype: int64
```

_Notice how you can now easily look up Bob's score just by using his name as the key, like this: `scores_series["Bob"]`._
