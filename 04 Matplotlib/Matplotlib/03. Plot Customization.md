### 1. Line Customizations

These parameters control the appearance of the line that connects your data points.

- **`color` (or `c`)**: Sets the color of the line. You can use common color names (`'red'`, `'blue'`), single-letter shorthands (`'r'`, `'b'`, `'g'`), or hex codes (`'#FF5733'`).
    
    - _Example:_ `color='green'`
        
- **`linestyle` (or `ls`)**: Changes the pattern of the line.
    
    - `'-'` : Solid line (default)
        
    - `'--'` : Dashed line
        
    - `':'` : Dotted line
        
    - `'-.'` : Dash-dot line
        
    - _Example:_ `linestyle='--'`
        
- **`linewidth` (or `lw`)**: Adjusts the thickness of the line. It takes a numerical value (float or integer).
    
    - _Example:_ `linewidth=2.5`
        

### 2. Marker Customizations

Markers are the visual symbols placed exactly where your data points lie on the graph.

- **`marker`**: Defines the shape of the data point. If you don't specify this, no markers will appear (just the line).
    
    - `'o'` : Circle
        
    - `'s'` : Square
        
    - `'^'` : Triangle
        
    - `'*'` : Star
        
    - `'D'` : Diamond
        
    - _Example:_ `marker='o'`
        
- **`markersize` (or `ms`)**: Controls how large the marker appears. It takes a numerical value.
    
    - _Example:_ `markersize=12`
        
- **`markerfacecolor` (or `mfc`)**: Sets the fill color _inside_ the marker. If left blank, it usually defaults to the line color.
    
    - _Example:_ `markerfacecolor='yellow'`
        
- **`markeredgecolor` (or `mec`)**: Sets the color of the border/outline around the marker.
    
    - _Example:_ `markeredgecolor='black'`

         color='purple',             # Line is purple

         linewidth=2,                # Line thickness is 2
         marker='D',                 # Markers are Diamonds
         markersize=14,              # Marker size is 14
         markerfacecolor='cyan',     # Inside of diamond is cyan
         markeredgecolor='black')    # Border of diamond is black

# Adding labels for clarity
plt.title("Matplotlib Customization Example")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")

# Display the plot
plt.grid(True)
plt.show()
